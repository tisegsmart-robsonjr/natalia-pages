Política de Privacidade

AutoPost Creator — Ferramenta de agendamento e publicação de conteúdo
Última atualização: 13 de março de 2026

1. Quem somos

AutoPost Creator é uma ferramenta de agendamento e publicação automatizada de conteúdo em redes sociais. Nosso objetivo é ajudar criadores de conteúdo, influenciadores e pequenas empresas a gerenciar suas publicações de forma prática e eficiente.

2. Dados que coletamos

Para o funcionamento da integração com o TikTok, coletamos e armazenamos:

  • Token de acesso OAuth2 fornecido pelo TikTok após autorização expressa do usuário
  • Token de atualização (refresh token) para renovação automática do acesso
  • Informações básicas do perfil TikTok autorizado (user.info.basic): nome de exibição e identificador de usuário

Não coletamos dados de visitantes do site, nem dados de terceiros além do estritamente necessário para a operação do serviço.

3. Finalidade do uso dos dados

Os dados coletados são utilizados exclusivamente para:

  • Publicação automatizada de conteúdo (imagens e vídeos) nos perfis TikTok autorizados
  • Autenticação e renovação automática do token de acesso à API do TikTok
  • Gerenciamento do calendário editorial dos usuários

Nenhum dado é utilizado para fins publicitários, venda a terceiros ou análise de comportamento.

4. Compartilhamento de dados

Não compartilhamos, vendemos ou transferimos dados pessoais para terceiros. Os tokens de acesso são armazenados de forma segura em nossa infraestrutura e utilizados exclusivamente para chamadas autorizadas à API do TikTok.

5. Retenção e exclusão

Os tokens de acesso são mantidos enquanto a integração estiver ativa. Para solicitar a exclusão, basta revogar o acesso do aplicativo nas configurações de privacidade do seu perfil TikTok (Configurações → Privacidade → Aplicativos com acesso). Após a revogação, todos os tokens associados são excluídos de nossos servidores em até 30 dias.

6. Segurança

Os tokens são armazenados em ambiente seguro com acesso restrito. Utilizamos criptografia em trânsito (HTTPS/TLS) e não transmitimos credenciais por canais não criptografados.

7. Seus direitos (LGPD)

Em conformidade com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018), você tem direito a:

  • Confirmação da existência de tratamento dos seus dados
  • Acesso aos seus dados tratados
  • Correção de dados incompletos, inexatos ou desatualizados
  • Exclusão dos dados tratados
  • Revogação do consentimento a qualquer momento
