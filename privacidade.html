Política de Privacidade

Natalia na Boleia — Projeto de conteúdo digital da Proteauto Brasil
CNPJ: 12.472.235/0001-65
Última atualização: 24 de fevereiro de 2026

1. Quem somos

Este aplicativo é uma ferramenta de automação de publicação de conteúdo nas redes sociais, operada internamente pela Associação dos Proprietários de Veículos Automotores do Brasil (Proteauto), para o projeto de conteúdo digital Natalia na Boleia.

2. Dados que coletamos

Para o funcionamento da integração com o TikTok, coletamos e armazenamos:

  • Token de acesso OAuth2 fornecido pelo TikTok após autorização expressa do usuário
  • Token de atualização (refresh token) para renovação automática do acesso
  • Informações básicas do perfil TikTok autorizado (user.info.basic): nome de exibição e identificador de usuário

Não coletamos dados de visitantes do site, nem dados de terceiros além do necessário para a operação da automação.

3. Finalidade do uso dos dados

Os dados coletados são utilizados exclusivamente para:

  • Publicação automatizada de conteúdo (imagens e vídeos) no perfil TikTok autorizado
  • Autenticação e renovação automática do token de acesso à API do TikTok

Nenhum dado é utilizado para fins comerciais, publicitários ou de análise de comportamento.

4. Compartilhamento de dados

Não compartilhamos, vendemos ou transferimos dados para terceiros. Os tokens de acesso são armazenados de forma segura na instância de automação (n8n) e utilizados exclusivamente para chamadas autorizadas à API do TikTok.

5. Retenção e exclusão

Os tokens de acesso são mantidos enquanto a integração estiver ativa. Para solicitar a exclusão, basta revogar o acesso do aplicativo nas configurações de privacidade do seu perfil TikTok (Configurações → Privacidade → Aplicativos com acesso).

6. Segurança

Os tokens são armazenados em ambiente seguro com acesso restrito à equipe operacional da Proteauto. Não são transmitidos via canais não criptografados.

7. Seus direitos (LGPD)

Em conformidade com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018), você tem direito a:

  • Confirmação da existência de tratamento dos seus dados
  • Acesso aos seus dados tratados
  • Correção de dados incompletos, inexatos ou desatualizados
  • Exclusão dos dados tratados
  • Revogação do consentimento a qualquer momento

Para exercer qualquer desses direitos, entre em contato pelo e-mail abaixo.
